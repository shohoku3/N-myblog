> A N-myblog 
基于nodeJS Express MongoDB 的个人博客

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
supervisor index

# mongoDB index DB 
mongod.exe --dbpath "path"

# run MongoDB 
mongod.exe

#run 
mongo.exe

#visit
localhost:3000/posts

```
